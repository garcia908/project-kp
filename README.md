## 🚀 Cara Pakai

1. Download semua file (index.html, style.css, script.js)
2. Simpan dalam satu folder
3. Double-click index.html ATAU
4. Pakai Live Server di VS Code

**Ubah Warna**: Edit `:root` di style.css
**Tambah Produk**: Copy product-card di HTML
